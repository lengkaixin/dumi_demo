# md_components

## Getting Started

由于该组件库是在 React 框架基础上对 Antd Design 进行的二次封装。

本 UI 库部分组件个性化，没个性化的都可以使用 antd 版本的组件，除了新增的组件属性，antd 的默认属性都兼容。

附 antd 官网地址：https://ant.design

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
